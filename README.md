# 智能刷题 (Smart Quiz App)

一款基于 React + Electron + Vite 开发的现代化桌面端智能刷题与题库管理软件。支持全题型识别、智能文件解析、艾宾浩斯记忆曲线复习模式以及沉浸式深色模式。

## 🌟 核心功能

*   **智能文件解析与导入**
    *   支持 `Excel` (`.xlsx`, `.xls`)、`CSV`、`JSON`、`TXT` 格式一键导入。
    *   **智能题型推断**：无需严格的固定格式，程序会自动通过 `___` 推断填空题、通过选项数量推断单/多选题，或通过对/错关键字推断判断题。
    *   支持单选题、多选题、判断题、填空题、简答题、编程题等全题型。
    *   内置各格式全题型“导入模板”一键下载功能。
*   **极致的性能体验**
    *   基于 Vite 代码拆分与按需加载，极速冷启动（解决白屏问题）。
    *   CSS 过渡动画极速响应优化，提供指哪打哪的清爽点击反馈。
*   **本地数据安全**
    *   采用 `sql.js` (SQLite WASM) 作为本地数据库。
    *   数据库文件存放在用户本地应用数据目录中，并使用 gzip 压缩算法优化存储占用，保护隐私且无需联网。
*   **科学复习与考试模式**
    *   **练习模式**：答题后立即显示正确答案与解析。
    *   **考试模式**：倒计时限制，最后统一交卷并查看成绩与报告。
    *   **错题本与收藏夹**：一键收藏重点难题，自动记录错题。
    *   **艾宾浩斯记忆法**：(Spaced Repetition) 根据每次答题的时间和正确率，智能安排下一次的最佳复习时间。
*   **高颜值界面**
    *   现代化的暗色系 UI (Dark Mode) 与玻璃拟态 (Glassmorphism) 风格设计。

## 🛠️ 技术栈

*   **前端框架**: React 18
*   **构建工具**: Vite 6
*   **桌面端引擎**: Electron 33
*   **UI 组件库**: Ant Design 5
*   **数据库**: sql.js (SQLite3 WebAssembly)
*   **图表库**: Recharts
*   **电子表格解析**: SheetJS (xlsx)

## 🚀 快速开始

### 1. 安装依赖

请确保您的电脑上已安装 [Node.js](https://nodejs.org/)。

```bash
npm install
```

### 2. 启动开发环境

```bash
npm run dev
```

### 3. 打包构建可执行文件 (EXE)

```bash
npm run build
```
执行完毕后，安装包将生成在 `release/` 目录下（如 `smart-quiz-app Setup 1.0.0.exe`）。

## 📁 目录结构简述

*   `electron/`: Electron 主进程与 Preload 脚本。
*   `src/pages/`: 核心业务页面（工作台、题库管理、刷题引擎、数据统计等）。
*   `src/db/`: 基于 `sql.js` 封装的本地数据库仓库与服务。
*   `dist/` & `dist-electron/`: Vite 编译后的前端与主进程产物。
*   `release/`: Electron-Builder 打包后的成品输出目录。
